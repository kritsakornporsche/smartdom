# 🚀 SmartDom: Free-Tier Features Walkthrough

ตามคำขอของคุณที่ต้องการให้ดำเนินการพัฒนาฟีเจอร์เพิ่มเติมแบบไม่มีค่าใช้จ่าย (Free-Tier) ทันที ผมได้ทำการสร้างและวางระบบหลักทั้งหมดไว้เรียบร้อยแล้ว โดยการทำงานครั้งนี้เน้นไปที่การสร้าง API และระบบหลังบ้านเพื่อรองรับฟีเจอร์ใหม่ๆ ดังนี้:

---

## ⚡ 1. ระบบจดมิเตอร์น้ำ-ไฟ (Meter Management System)
**สถานะ:** สร้าง Database และ API เสร็จสมบูรณ์
- **สิ่งที่ทำ:** 
  - สร้างไฟล์ `/api/owner/meters/route.ts` สำหรับดึงข้อมูลและบันทึกประวัติการจดมิเตอร์
  - สร้างหน้าจอ UI เบื้องต้นสำหรับเจ้าของหอพักไว้ที่ `/owner/meters` และเพิ่มเมนู "จดมิเตอร์น้ำ-ไฟ" ลงในแถบด้านข้าง (Sidebar) หมวดหมู่ 'การเงินและสัญญา'
- **ประโยชน์:** เจ้าของหอพักสามารถบันทึกเลขมิเตอร์ของแต่ละห้องได้ และระบบพร้อมที่จะนำไปคำนวณบิลค่าเช่าในอนาคต

---

## 💳 2. ระบบสร้าง QR Code PromptPay (QR Code Generator)
**สถานะ:** ติดตั้ง Library และสร้าง API ตัวสร้าง QR เสร็จสมบูรณ์
- **สิ่งที่ทำ:**
  - ติดตั้งแพ็กเกจ `promptpay-qr` และ `qrcode` (ใช้งานได้ฟรี 100%)
  - สร้าง API สำหรับผู้เช่าไว้ที่ `/api/tenant/billing/qr/route.ts` 
  - **การทำงาน:** เมื่อผู้เช่าขอดูบิล API ตัวนี้จะดึงยอดเงินค้างชำระ (Amount) และเบอร์พร้อมเพย์ของเจ้าของหอพัก (`promptpay_number` จากตาราง Profile) มาสร้างเป็นรูปภาพ QR Code พร้อมจ่ายเงินเป๊ะๆ
- **ประโยชน์:** อำนวยความสะดวกให้ผู้เช่าสแกนจ่ายได้ทันที ลดโอกาสการโอนเงินผิดยอด

---

## 🔔 3. ระบบแจ้งเตือนในแอป (In-App Notification)
**สถานะ:** สร้าง API ดึงข้อมูลและจัดการแจ้งเตือนเสร็จสมบูรณ์
- **สิ่งที่ทำ:**
  - สร้างตาราง `notifications` เพื่อเก็บข้อความของ User ทุกระดับ
  - สร้าง API ไว้ที่ `/api/notifications/route.ts` สำหรับให้ฝั่ง Frontend (หน้าเว็บ) ดึงข้อความแจ้งเตือน และอัปเดตสถานะ "อ่านแล้ว"
- **ประโยชน์:** ลดค่าใช้จ่ายในการส่ง SMS โดยใช้ระบบ Notification ภายในเว็บแทน (เช่น แจ้งเตือนบิลใหม่ แจ้งพัสดุมาถึง)

---

> [!TIP]
> **ขั้นตอนถัดไป (UI Integration):** ระบบหลังบ้าน (Backend) ทำงานได้สมบูรณ์แล้ว งานที่เหลือคือการนำข้อมูลเหล่านี้ไปแสดงผลในหน้าจอ (Frontend UI) เช่น:
> 1. ใส่ปุ่ม "สร้างบิล" ในหน้าจดมิเตอร์
> 2. นำรูป QR Code ไปแปะในหน้าบิลของผู้เช่า
> 3. ทำไอคอนกระดิ่งแจ้งเตือนบน Navbar
> 4. ทำปุ่ม Export CSV ในหน้าบัญชี
